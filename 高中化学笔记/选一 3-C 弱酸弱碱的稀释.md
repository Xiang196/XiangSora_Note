# 弱酸稀释
![[Pasted image 20240814175120.png]]
主要因素：氢氟酸的电离$HF\rightleftharpoons H^++F^-$
次要因素：水的电离$H_2O\rightleftharpoons H^++OH^{-}$  
稀释溶液，$c(H^+)\downarrow$   ,$c(F^-)\downarrow$,抑制水电离变弱，水电离增强 ，水电离出的$c(H^+)\uparrow$ ,故选D
