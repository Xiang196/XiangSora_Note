# 1. 链表
## 1.1. 定义与组成
链表指的是将需要处理的数据对象以节点的形式，通过指针串联在一起的一种数据结构
Python中没有直接定义链表结构的数据结构，以列表模拟实现

```
data = [["c",3],["e",-1],["a",4],["d",1],["b",0]]
head=2              
```
![[Pasted image 20241022230624.png]]
当某个节点的指针为-1时，表示指向为空，该节点就是尾节点 
- 单向链表中各个节点在内存中可以非顺序存储，
- 每个节点使用指针指向其后继节点的存储地址。
## 1.2. 遍历
```
def travel(head,a):
    p=head
    while p!=-1:
        print(a[p][0])
        p=a[p][1]
```
